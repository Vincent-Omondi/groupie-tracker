What is Groupie Tracker

Groupie Trackers consists on receiving a given API and manipulate the data contained in it, in order to create a site, displaying the information.

What does the project entail

It will be given an API, that consists in four parts:

  1. The first one, artists, containing information about some bands and artists like their name(s), image, in which year they began their activity, the date of their first album and the members.
  2. The second one, locations, consists in their last and or upcoming concert locations.
  3. The third one, dates, consists in their last and/or upcoming concert dates.
  4. The last one, relation, does the link between all the other parts, artists, dates and locations.

What does the project help one learn about

The project helps one learn about:

  1. Manipulation and storage of data.
  2. JSON files and format.
  3. Event creation and display.
  4. Client-server.
